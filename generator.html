Generator Link Undangan

Dika & Lisa — Buat link undangan dan link WhatsApp untuk setiap tamu

Panduan Penggunaan

  1. Download Template — klik tombol Download Template untuk mendapatkan file Excel.
  2. Isi Data — buka file Excel, isi kolom Nama (wajib) dan Nomor WA (opsional). Satu baris = satu tamu.
  3. Import Excel — klik input file Choose File, pilih file Excel/CSV yang sudah diisi. Data akan masuk otomatis.
  4. Edit Template Pesan — sesuaikan pesan WA di bagian bawah. Gunakan [NAMA] untuk nama tamu dan [LINK] untuk link undangan.
  5. Salin Template Pesan — klik tombol untuk menyalin template pesan saat ini.
  6. Generate Link — klik Generate Link untuk membuat link undangan dan pesan WA per tamu.
  7. Kirim WA — klik Kirim WA di setiap baris untuk langsung membuka WhatsApp.
  8. Salin Link / Salin WA — klik untuk menyalin link undangan atau link WA.
  9. Export CSV — klik Export CSV untuk mengunduh hasil generate dalam format CSV.
  10. Reset Data — klik jika ingin mengganti data import atau memulai dari awal. Sebisa mungkin lakukan import hanya 1x untuk menghindari duplikasi dari import sebelumnya.
Data ini diisi otomatis dari import Excel.
Upload file Excel (.xlsx, .xls, .csv) dengan kolom Nama dan Nomor WA. Kosongkan nomor jika tidak ada.
Gunakan placeholder [NAMA] untuk nama tamu dan [LINK] untuk link undangan.
